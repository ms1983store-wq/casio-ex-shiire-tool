# Casio EX 仕入れ判定ツール

メルカリの公開検索結果をもとに、Casio EXILIM EXシリーズの型番、カラー、仕入れ金額から利益額、利益率、仕入れ判定を計算する静的Webアプリです。

## 使い方

`index.html` をブラウザで開くだけで使えます。型番、カラー、相場モードを選ぶと販売見込みが自動入力され、仕入れ金額を入れると判定が更新されます。

## 相場条件

- 確認日: 2026-05-28
- 対象: メルカリ公開検索結果、デジタルカメラカテゴリ相当、Casio EXILIM EX-ZR系を中心としたEX型番
- 価格: 各型番の表示価格帯から作成した参考中央値
- 状態: 未使用に近い、目立った傷や汚れなしを「目立った傷なし」側に集約
- ジャンク: タイトルにジャンク、訳あり、難あり、故障、動作未確認などを含むものを集計
- 判定条件、送料、手数料、UI、公開構成は既存の仕入れ判定アプリと同じ構成

## ファイル

- `index.html`: 画面
- `styles.css`: スタイル
- `app.js`: 相場データと計算ロジック
